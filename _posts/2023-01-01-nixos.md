## Modules

### Structure

```nix
{
  imports = [
    # paths to other modules
  ];

  options = {  };

  config = {  };
}
```

#### shorthand

```nix
{
  imports = [
    # paths to other modules
  ];

  # option definitions
  # ...
}
```


### Eval

```nix
:l <nixpkgs.lib>

m1 = { options.foo = mkOption {}; }

m2 = { foo = "hello" }

m3 = { foo = "world" }

(evalModules { modules = [ m1 m2 m3 ]; }).config.foo

#> ["world" "hello"]
```
