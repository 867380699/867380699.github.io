# Hotkeys

- G - Move (Grab)
- S - Scale
- R - Rotate

> X Y Z and Scroll Button

- Mid Mouse Button - Orbiting
- Shift + Mid Mouse Button - Pan
- Scroll wheel - Zoom
- Numpad `.` - Focus
- Tilda - Focus 


- X - Delete
- Shift + A - Add


F3 - search

Tab - Switch Object Mode/Edit Mode



- A - select all
- alt + A - deselect all
