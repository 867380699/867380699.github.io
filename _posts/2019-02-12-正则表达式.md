# 分组、引用
`(ABC)` - capturing group
`(?:ABC)` - non-capturing group
`\1` - numeric reference

# Lookaround
`(?=ABC)` - positive lookahead
`(?!ABC)` - negative lookahead

# 限定符
`+` - 1 or more
`*` - 0 or more
`?` - optional
`{1,3}`

`?` - lazy 

# 替换
`$&` - match
`$1` - capture group

# Flags
`i` -  case **i**nsensitive
`g` - **g**lobal search
`m` - **m**ultiline
