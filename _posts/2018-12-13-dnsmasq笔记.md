# 安装
## Mac
```
brew install dnsmasq
```
## ubuntu
```
sudo apt-get install dnsmasq
```

# 配置
`/usr/local/etc/dnsmasq.conf`

```conf
# DNS绑定地址
listen-address=192.168.1.102,127.0.0.1
# 配置解析顺序文件
resolv-file=/usr/local/etc/resolv.dnsmasq.conf
```
