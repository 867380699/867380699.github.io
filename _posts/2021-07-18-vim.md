# vim

`vim` - Vi IMproved, a programmer's text editor

## Move
Basic - `h` `j` `k` `l`
Word - `w` `b` `e` `ge`
Line - `^` `$` `0` `j` `g` `G`
Paragraph - `{` `}`
Screen - `Ctrl+d` `Ctrl+u` `Ctrl+f` `Ctrl+b`
Char - `f` `F` `t` `T` `;`
Parenthesis - `%`

## Search
`/` `?` `n` `N`
`*` `#`

## Insert
`i` `a` `o`
`I` `A` `O`

## Edit
`dd` `yy` `p`
`x` `u`

`v` `V`

## .vimrc

```r
set number
set expandtab
set tabstop=2
set softtabstop=2
syntax on
set hlsearch
```

## vim-plug

```r
call plug#begin('~/.vim/plugged')

Plug 'vim-airline/vim-airline'
Plug 'preservim/nerdtree'
Plug 'jistr/vim-nerdtree-tabs'

call plug#end()
```
