# Functions

- `sh` - shell command
- `gv` - global variable
- `dp` - date parser
  + `$dp{0h0m0s}` - today's midnight
- `df` - date format
  + `df(EEE, a1d)` - tomorrow's short day name
  + `df(o)` - days in current month 
- `wg` - web get
- `si` system info
  + `si(mindex, 1)` - module index in it's parent's parent
