```bash

# or
choco find tree
```

```bash

```

## FluentTerminal

### 配置Git Bash
可执行程序位置设置为

```
C:\Program Files\Git\bin\bash.exe
```

修改 `.bashrc` 支持中文显示

```bash
export LANG="zh_CN.UTF-8@cjknarrow"
```
